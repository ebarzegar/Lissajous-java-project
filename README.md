# Lissajous Curve Visualizer (Java)

This project is a Java Swing application that visualizes Lissajous curves based on user-defined parameters.

## Features
- Interactive GUI using Java Swing
- User input for parameters (a, b, delta)
- Real-time curve rendering
- Mathematical curve generation using sine functions

## Concepts Used
- Event-driven programming (ActionListener)
- Graphics rendering (paintComponent)
- Java Swing (JFrame, JPanel, JTextField)
- Basic algorithms (GCD calculation)
